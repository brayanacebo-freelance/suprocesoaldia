Helper Style

CSS helper contains short code (such as background, color, font size, box corner, etc) that can be used to support existing features. You can add your css here or create a new css file.

Combine the helper

Helper element supports to be used together. It helps to condense the code that we created.

<div class="demo-box grd-red color-white corner-all">helper</div>
<div class="demo-box grd-orange color-white corner-tl corner-br shadow">helper</div>
<div class="demo-box bg-transparent shadow-black"><span class="bold italic underline">helper</span></div>
<div class="demo-box bg-teal color-white corner-tr corner-bl">helper</div>
