Overview

Stilearn admin designed to have three columns (left side, content, right side). But, this can also be made ​​into two columns just by removing the right column.

Developed on the basis of bootstrap twitter, if you want to see the documentation please visit the twitter bootstrap. We have not changed any of their code, so if you want to update the version of the bootstrap, you just need to overwrite the old version of the file.

Requires HTML5 doctype

Stilearn makes use of certain HTML elements and CSS properties that require the use of the HTML5 doctype. Include it at the beginning of all your projects.

<!DOCTYPE html>
<html lang="en">
  ...
</html>

Integration

Need to remember, to use stilearn feature you must include at least the following elements:

  1. For CSS code, you must include : 
    • <link href="css/bootstrap.css" rel="stylesheet">
<link href="css/bootstrap-responsive.css" rel="stylesheet">
<link href="css/stilearn.css" rel="stylesheet">
<link href="css/stilearn-responsive.css" rel="stylesheet">
<link href="css/stilearn-helper.css" rel="stylesheet">

      Other css (animate, fonts, plugins, etc) according to your case.

  2. For JS code, you must include : 
    • <!--required plugin template-->
<script type="text/javascript" src="http://platform.twitter.com/widgets.js"></script>
<script src="js/jquery.js"></script>
<script src="js/bootstrap.js"></script>

<!--required stilearn template-->
<!--you must include this file. you can edit the code in it, but be careful.-->
<script src="js/stilearn-base.js"></script>

<!--this required only to demo, you can remove it.-->
<script src="js/holder.js"></script>

      Other js (plugins, etc) according to your case.

HTML Template

To create a simple display, we'll utilize a basic HTML template that includes everything mentioned above.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Stilearn Admin Bootstrap</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="">
    <meta name="author" content="stilearning">
    
    <!--google font-->
    <link href="http://fonts.googleapis.com/css?family=Aclonica:regular" rel="stylesheet" type="text/css">

    <link href="css/bootstrap.css" rel="stylesheet">
    <link href="css/bootstrap-responsive.css" rel="stylesheet">

        <div class="row-fluid">
            <!-- span side-left -->
            <div class="span1">
                <!--side bar-->
                <aside class="side-left">

                    <div class="content-header">
                    ...

                        <div class="sidebar-right-control">
                        ...
                        </div><!-- /sidebar-right-control-->

                    </div><!-- /sidebar-right -->
