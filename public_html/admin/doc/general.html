General

General style is css that governs all stilearn features (such as widgets, side bar menu, layout, chat, etc), there are in the file stilearn.css. I'll just give an example of implementation of the widget. For other elements, please see the documents demo, you just need to copy and edit it.

Widget

Some features such as widgets, requires a bit of javascript code. We've included all of them in the file stilearn.base.js. So you just need to write html tags and everything will work just like the demo.

<div class="box corner-all">
    <div class="box-header grd-white color-black corner-top">
        <div class="header-control">
            <a data-box="collapse"><i class="icofont-caret-up"></i></a>
            <a data-box="close">&times;</a>
        </div>
        <span>Header Gradient White</span>
    </div>
    <div class="box-body">
        <p>Lorem ipsum dolor sit amet, natoque per at morbi at vestibulum leo.</p>
    </div>
</div>

You can combine with css helpers, such as background, corner, font-color, etc. or other css, such as css-animate.
